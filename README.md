### pnpm

```bash
pnpm create hexo [path]

# pnpm dlx create-hexo [path]
```

> [pnpm.io](https://pnpm.io/)

### npm

```bash
npm init hexo [path]

# npm exec create-hexo [path]
# npx create-hexo [path]
```

> [npm init](https://docs.npmjs.com/cli/commands/npm-init)

### Yarn

```bash
yarn create hexo [path]
```
